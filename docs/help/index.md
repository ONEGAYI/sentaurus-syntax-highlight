# Sentaurus TCAD Syntax 帮助文档

> 欢迎使用 Sentaurus TCAD Syntax 扩展！本文档将帮助你快速了解扩展功能。

## 功能概览

### 语法高亮

支持 7 种 Sentaurus TCAD 工具语言的语法高亮：

| 语言 | 工具 | 文件模式 |
|------|------|----------|
| SDE | Structure Editor | `*_dvs.cmd`, `.scm` |
| SDEVICE | Device Simulator | `*_des.cmd` |
| SPROCESS | Process Simulator | `*_fps.cmd`, `.fps` |
| EMW | EM Wave | `*_eml.cmd`, `*_emw.cmd` |
| Inspect | Inspect | `*_ins.cmd` |
| Svisual | Sentaurus Visual | `*_vis.tcl` |
| SDEVICE PAR | Device Material DB | `*.par` |

### 自动补全

- 关键词补全（基于官方手册提取）
- 用户变量补全与跳转
- 函数签名提示（SDE）
- 上下文感知补全（SDEVICE section 内）

### 代码片段

通过命令面板 `Sentaurus: Insert Snippet` 快速插入预定义代码模板。
