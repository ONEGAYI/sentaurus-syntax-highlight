# Sentaurus TCAD Syntax Help

> Welcome to the Sentaurus TCAD Syntax extension! This document will help you get started with the extension features.

## Feature Overview

### Syntax Highlighting

Supports syntax highlighting for 7 Sentaurus TCAD tool languages:

| Language | Tool | File Pattern |
|----------|------|--------------|
| SDE | Structure Editor | `*_dvs.cmd`, `.scm` |
| SDEVICE | Device Simulator | `*_des.cmd` |
| SPROCESS | Process Simulator | `*_fps.cmd`, `.fps` |
| EMW | EM Wave | `*_eml.cmd`, `*_emw.cmd` |
| Inspect | Inspect | `*_ins.cmd` |
| Svisual | Sentaurus Visual | `*_vis.tcl` |
| SDEVICE PAR | Device Material DB | `*.par` |

### Auto Completion

- Keyword completion (extracted from official manuals)
- User variable completion and go-to-definition
- Function signature help (SDE)
- Context-aware completion (within SDEVICE sections)

### Snippets

Quickly insert predefined code templates via Command Palette `Sentaurus: Insert Snippet`.
